## Overview

Panel attack is a tetris Attack/Panel de Pon clone in lua made with LOVE

## Dependencies

* Love 0.9.0
* Lua 5.3

## Running

* Clone the repo

```git clone https://github.com/TecProg-grupo4-2018-2/panel-attack.git```

* cd into the cloned repo

```cd panel-attack```

* Run the game with LOVE

```love ./```
